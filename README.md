# EcoSim
------

Модель трехуровневой экосистемы, реализующая эволюцию с помощью генетического алгоритма и машинного обучения.
 
Обучение нейросети реализовано с помощью модифицированного алгоритма NEAT. 

На текущий момент в процессе обновления, система сохранений не будет обрабатывать некоторые гены и сущности.
